***LEVANTAMENTO DE REQUISITOS:***



***Funcionais:***



**RF01:** o sistema deve permitir que os professores consigam lançar dados do aluno, como nome, classe e notas.



**RF02:** O sistema deve permitir que sejam feitos cálculos da média das notas do aluno.



**RF03:** Após o cálculo das médias, o sistema deve informar se o aluno está aprovado ou reprovado.



**RF04:** O sistema deve gerar um relatório com as informações dos alunos.







***NÃO FUNCIONAIS:***



**RNF01:** O sistema deve manter segurança dos dados.



**RNF02:** O sistema não deve demorar para processar os dados.



**RNF03:** O sistema deve tratar os erros de dados inválidos.







***REGRAS DE NEGÓCIO:***



**RN01:** Apenas professores e equipe de gestão do SENAI tem acesso ao sistema.



**RF02:** Apenas professores podem lançar as notas dos alunos.
